# Number Guessing Game

A simple command-line number guessing game built in Java.  
The program generates a random number and prompts the user to guess it, giving feedback on each attempt.    
  This project is built for [roadmap.sh submission](https://roadmap.sh/projects/number-guessing-game)

## Requirements
- Java 8 or newer

## How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/adriantriputra/number-guessing-game.git.
   ```
2. Navigate to the project folder:
   ```bash
   cd number-guessing-game
   ```
3. Compile the program:
   ```bash
   javac Main.java
   ```
4. Run it:
   ```bash
   java Main
   ```
